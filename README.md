## `> cat pipeline.security.yml`

```yaml
# ┌─────────────────────────────────────────────────────────────────┐
# │                🔐 SECURE CI/CD PIPELINE BLUEPRINT               │
# └─────────────────────────────────────────────────────────────────┘

on: [push]

jobs:

  pre_commit:
    runs: [gitleaks, detect-secrets, commitlint]
    blocks_merge_if: secrets_detected == true

  static_analysis:
    sast:  semgrep --config=auto
    sca:   snyk test && trivy fs .
    lint:  bandit -r . (Python security linter)

  infrastructure_security:
    iac_scan:       checkov -d ./terraform
    secrets_scan:   trufflehog --regex .
    policy_check:   OPA (Open Policy Agent)

  container_security:
    image_scan:     trivy image $IMAGE
    runtime_policy: falco --rules devsecops.yaml
    signing:        cosign sign $IMAGE

  dynamic_analysis:
    dast: owasp-zap -t $STAGING_URL
    api:  42crunch / swagger-ui security audit

  deploy:
    strategy:       blue_green
    zero_trust:     enforce_always
    secrets_mgmt:   AWS Secrets Manager / Vault

  post_deploy:
    siem_alert:     wazuh → elastic → pagerduty
    cspm_scan:      AWS Security Hub / Prisma Cloud
    auto_remediate: lambda_responder.py
```
